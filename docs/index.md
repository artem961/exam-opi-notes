# Конспект к экзамену

Сборник билетов. Навигация — в сайдбаре слева, поиск — сверху
(работает по русским словоформам).

!!! tip "Как читать"
    Билеты упорядочены по номеру. Определения и частые ошибки вынесены в
    цветные плашки — удобно для быстрого повторения.

- [ ] Билеты 1–20
- [ ] Билеты 21–40
- [ ] Билеты 41–60
- [ ] Билеты 61–80
